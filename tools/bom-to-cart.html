BOM-to-Cart

Parse any BOM, search real-time Octopart pricing, and compute the cheapest purchasing plan. The AI agent handles the thinking; the extension handles the clicking.

OpenRouter-powered agent Octopart search workflow Top 5 optimized plans

What it does

  1. Parse any BOM pasted as markdown, CSV, or plain text into a normalized table.
  2. Search Octopart across distributors like DigiKey, Mouser, Newark, Arrow, and more.
  3. Optimize the part list into the cheapest shopping plan while balancing parts cost and shipping.

Architecture

Chrome Extension (Popup Dashboard)
  ├─ BOM table
  ├─ Best plans
  └─ Search console
        │
        ├─ Background service worker
        │    └─ OpenRouter API agent
        │
        └─ Content scripts
             └─ Octopart.com interactions

The agent calls OpenRouter directly, so there is no Hermes gateway dependency. Each invocation is short-lived and stateless.

Installation

  1. Clone the repo or download the extension/ directory.
  2. Open chrome://extensions/ and enable Developer mode.
  3. Click Load unpacked and select the extension/ folder.
  4. Open Options and set your OpenRouter API key.
  5. Click the extension icon to open the dashboard.

Limitations

  • Octopart CAPTCHA interrupts automated search after a handful of parts.
  • Generic parts like screws, nuts, and wire may not be indexed.
  • Shipping defaults to $10 per platform unless you override it in the optimizer.
